# VoxMate Privacy Policy

VoxMate collects and processes only the minimum information required to operate the extension inside Zoho Cliq.  
This includes:

- Zoho user ID (for task creation, reminders, and responses)
- Messages sent to the VoxMate bot (to generate replies)
- Tasks/reminders/events created through the extension
- No sensitive personal data is collected or stored.

## Data Storage

VoxMate stores data only in your connected backend (local server / ngrok / your own database).  
No external third-party service receives or processes your data except AI APIs explicitly configured by you.

## Data Sharing

VoxMate does not share or sell any user data.  
Data is used only for performing the requested actions like reminders, tasks, events, or general chat.

## User Control

You may uninstall the extension anytime, which stops all data processing instantly.

## Contact

For any issue, contact the developer of this extension.
